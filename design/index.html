Typography

<Text>This is normal text.</Text>This is normal text.

Always wrap text in a <Text> component. This way basic styling is applied and default browser values get overridden.

<Text variant="h1">This is a page title.</Text>

This is a page title.

<Text variant="h2">This is a section title.</Text>

This is a section title.

<Text variant="small">This is small text.</Text>This is small text.<Text variant="code">This is code.</Text>This is code.

Use the variant prop to apply pre-defined styles to your text.
